Phát triển Solution

1. Cấu trúc

Cấu trúc folder của một solution bao gồm:

1 solution.php Bắt buộc Đây là file php khai báo thông tin của solution, sử dụng cho việc cài đặt và quản lý solution
2 plugin Tùy chọn Đây là folder plugin nằm trong solution. Hướng dẫn phát triển plugin

2. Install / Uninstall

Để cài đặt được solution cần tối thiếu 1 file solution.php:

File solution.php example 
<?php
    return [
        'tags' => [],
        'type' => 'solution',
        'solution' => 'new_solution',
        'folder_name' => 'new_solution',
        'name' => 'New Solution',
        'dependencies' => []
    ];

Trong đó:

- tags: Các thẻ của solution

- type: solution, thông tin này không thay đổi

- solution: new_solution, thông tin bắt buộc

- folder_name: là tên folder của solution

- name: tên solution

- dependencies: Chứa thông tin danh sách các plugin hoặc solution cần cài đặt trước khi cài đặt solution. Ví dụ solution phụ thuộc 

'dependencies' => [
        'solutionA',
        'solutionB/pluginC'
    ]

Việc Install / Uninstall solution sẽ dựa trên thông tin của file solution.php để thực hiện.
